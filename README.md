# mcp-confluence-demo

A demonstration project for MCP (Model Context Protocol) with Confluence integration.

## Prerequisites

This project uses [mise](https://mise.jdx.dev/) for tool version management.

## Setup

Install required tools:

```bash
mise install
```

## Tools

- Node.js: 22.13.1
- pnpm: 10.11.1
- bats: 1.11.1
- shellcheck: 0.10.0
- actionlint: 1.7.7
- github-cli: 2.74.1
- ripgrep: 14.1.1
